# redux-jquery
- Redux+jQuery samples
- ES2015(babel)
- **This sample don't use React**
 - You can learn focus on redux

## build
```
git clone https://github.com/mkamakura/redux-jquery.git
cd redux-jquery && npm i
npm start
```
