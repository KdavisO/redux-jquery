# redux-jquery
- Redux+jQuery exsamples
- ES2015(babel)
- **This examples don't use React**

## build
```
git clone https://github.com/mkamakura/redux-jquery.git
cd redux-jquery && npm i
npm start
```
Lunch local server and wachify.
